## 📖 About
 
<h3>Spend Jorge Lemann's Fortune</h3>

A fully responsive and fun web app that "allows" users to spend the entire fortune of the richest man in Brazil.
Project Inspired by [neal.fun](https://neal.fun)

## 🚀 Technologies 

- [HTML](https://html.com/)
- [CSS](https://developer.mozilla.org/pt-BR/docs/Web/CSS)
- [JavaScript](https://www.javascript.com/)
- [Bootstrap](https://getbootstrap.com/)

## ✨ Features

- Cart System<br/>
- JSON API consumption<br/>

## 🤔 How to use

- Clone this repo: `git clone https://github.com/michaelsribeiro/spendGameJS.git`

- Then go to the project's folder: `cd spendGameJS`

- `Live Server VsCode Extension needed`

- Open your browser in: `http://localhost:5500/`
